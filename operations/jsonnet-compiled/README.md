# jsonnet-compiled

Example Kubernetes manifests to deploy Tempo using Jsonnet based on different static configurations. Modify and regenerate to suite your needs.

### Build

To generate the manifests from this directory, run `make all`.

This requires [jsonnet-bundler](https://github.com/jsonnet-bundler/jsonnet-bundler) and [Tanka](https://tanka.dev/) to be installed.
