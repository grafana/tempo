# Grafana tempo-distributed Helm chart documentation

{{< admonition type="note" >}}
The `tempo-distributed` Helm chart is now maintained by the community. 
The chart has moved to the [grafana-community/helm-charts](https://github.com/grafana-community/helm-charts/tree/main/charts/tempo-distributed) repository.
For support and contributions, refer to the [community Helm charts repository](https://github.com/grafana-community/helm-charts).
{{< /admonition >}}

The tempo-distributed Helm chart for Grafana Tempo allows you to configure, install, and upgrade Grafana Tempo within a Kubernetes cluster. The chart is maintained by the community.
